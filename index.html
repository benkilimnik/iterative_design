Introduction

In this project, I set out to compare two different versions of an online cacti shop. With statistical tests in-hand, I examined whether interface A with a horizontally stacked layout is better or worse than interface B with a vertically stacked layout.

The two key metrics I examine are time to completion, which is the time it takes for a user to order cacti, and return rate, which is whether a user returns to the page or not after buying cacti.

Hypotheses

Time to Completion

Null hypothesis: Both interfaces A and B take equal or similar amounts of time to order cacti from.

Alternative hypothesis: Interface A takes longer to order because the layout is less symmetrical and less structured. Each image is stacked horizontally such that large images take up more vertical space. This makes it harder to find the order button because it is hidden beneath the image on different positions on the page. Interface B, on the other hand, features a more structured layout with each "Add to Cart" button place along the same vertical line.

Return Rate

Null hypothesis: Both interfaces have equal or similar return rates.

Alternative hypothesis: Interface B causes users to return more often to the page because it has more visual symmetry, making it more appealing to scroll down and browse through the cacti options.

Collecting Data

In order to collect data, I deployed a website on heroku that alternates between the two interfaces. When a user navigates to the page, there is a 50% chance of them seeing version A and a 50% chance of them viewing version B. This website was then shown to fellow students in my class, who purchased (fictional) cacti through the website.

Conclusion

  • limitations include a lack of data (only 17 distinct users were recorded in the heroku logs). This could be expanded by extending the heroku saved logs above 1500 lines.

  • Another limitation is that all users were of roughly the same age (all were college students). Extending user testing to a more diverse range of users could improve the reliability and applicability of the results, since college students may not be representative of all online shoppers for cacti.

  • Given inconclusive (statistically insignificant) results, we cannot make any inferences about design principles, as it is unclear which interface ultimately performed better.
